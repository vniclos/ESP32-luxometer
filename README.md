# ESP32 BH1750 LUXOMTER 

![alt text](https://github.com/vniclos/ESP32-luxometer/blob/master/images/esp32-bh1750_luxometer.jpg?raw=true "esp32 luxometer" )

## Hardware

- 1 ESP32 DEVKIT V1
- 1 BH1750

## Output 

### Wiring
![alt text](https://github.com/vniclos/ESP32-luxometer/blob/master/images/esp32-bh1750_luxometer.jpg?raw=true "esp32 luxometer" )

Connections:
  - BH1750 -> ESP32 
  - VCC -> 3V3 
  - GND -> GND
  - SCL -> SCL 
  - SDA -> SDA 
  - ADD -> NC

## Software
- Arduino IDE
- Library BH1750.h
