# ESP32 BH1750 LUXOMTER 

![alt text](https://github.com/vniclos/robot-arm4-esp32/blob/master/img/robot-00.jpg "Robot ARM4X builded" )

## Hardware

- 1 ESP32 DEVKIT V1
- 1 BH1750

. 
### Wiring

Connections:
  - BH1750 -> ESP32 
  - VCC -> 3V3 
  - GND -> GND
  - SCL -> SCL 
  - SDA -> SDA 
  - ADD -> NC

## Software
- Arduino IDE
- Library BH1750.h
